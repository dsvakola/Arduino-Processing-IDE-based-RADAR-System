Arduino + Processing RADAR System (v1.0)

This project demonstrates a RADAR-like scanning system using Arduino UNO, HC-SR04 ultrasonic sensor, and a servo motor. The graphical output is displayed using Processing IDE.

Files Included:
1. arduino_radar.ino – Arduino source code
2. arduino_radar_processing_code.pde – Processing visualization code

Setup:
- Upload the Arduino code.
- Open the Processing file and select the correct serial port (Line 25).
- Run the Processing sketch to view the RADAR output.

Hardware:
- Arduino UNO
- HC-SR04 ultrasonic sensor (Trig: D9, Echo: D10)
- SG90 servo motor (Signal: D3)
